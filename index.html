Trabalho 1 - Computação em nuvens -



O que é a Computação na Nuvem?

A computação em nuvem é um modelo de fornecimento de recursos de TI pela internet. Em vez de usar hardware e software locais, você pode acessar servidores, armazenamento, bancos de dados, redes, software e outros serviços por meio da internet, geralmente de forma sob demanda e escalável.

Quais as características da computação em Nuvem?

O usuário pode acessar os serviços sem ter ele localmente. Rapidez O usuário pode acessar serviços de computação via nuvem quando necessário, sem interagir com o provedor de serviços.

Quais as modalidades de serviços?

Infraestrutura como Serviço (IaaS): Oferece recursos de computação básicos, como servidores virtuais, armazenamento e redes.
Plataforma como Serviço (PaaS): Fornece um ambiente de desenvolvimento e implantação de aplicativos sem a necessidade de gerenciar a infraestrutura subjacente.
Software como Serviço (SaaS): Disponibiliza aplicativos completos pela internet, que você pode acessar através de um navegador.
Armazenamento na Nuvem: Permite o armazenamento de dados em servidores remotos, acessíveis pela internet.
Escalabilidade: A capacidade de ajustar automaticamente os recursos de acordo com a demanda. Isso significa que você pode aumentar ou diminuir a capacidade de processamento e armazenamento conforme necessário, sem precisar investir em hardware adicional.
Pagamentos Baseados em Uso: Muitos serviços de nuvem funcionam com um modelo de pagamento conforme o uso, o que significa que você paga apenas pelos recursos que utiliza, em vez de investir em infraestrutura fixa.
Acesso Global e Mobilidade: Com a nuvem, você pode acessar seus dados e aplicativos de qualquer lugar com conexão à internet, promovendo flexibilidade e colaboração remota.

Quais os tipos de nuvem? Defina cada uma delas.

Nuvem pública - Uma nuvem pública é um modelo de TI em que os provedores de serviços de nuvem pública disponibilizam serviços de computação, incluindo computação e armazenamento.
Nuvem privada - São serviços de computação em nuvem oferecidos pela internet ou por uma rede privada, somente a pessoas autorizadas, e não ao publico geral.
Nuvem híbrida - É um ambiente de computação misto em que aplicativos são executados usando uma combinação de computação, armazenamento e serviços em diferentes ambientes.

AWS




A AWS ( Amazon web serviços) foi o primeiro grande provedor de serviço de nuvens, atualmente possui o maior número de usuários. Também considerada o servidor mais caro, quando comparado ao Google cloud e Azure. A AWS possui grande integração IoT, IA e Machine Learning. Os serviços oferecidos pela AWS são diversos, como: 1 - Computação (criação de maquinas virtuais, computação serverless, gerenciamento e implantação de serviços web) 2 -serviço de armazenamento (Armazenamento de longo prazo para arquivamento e backup com baixo custo, Armazenamento de bloco para uso com instâncias EC2) 3 - Serviço de banco de dados 4 - Rede (Serviço de DNS escalável e altamente disponível,Conexão privada entre a AWS e o datacenter ou ambiente on-premises) 5 - Segurança (Controle de acesso e gestão de identidade, Gerenciamento de chaves de criptografia) 6 - Analytics e Big Data ( Processamento de grandes volumes de dados usando frameworks como Hadoop e Spark, Serviço de consulta SQL para dados armazenados no S3, Serviço para processamento de dados em tempo real) 7 - I.A e Machine Learning ( Plataforma para construir, treinar e implantar modelos de machine learning, Serviço de análise de imagens e vídeos, Serviço de processamento de linguagem natural (NLP)) 8 - IoT (Plataforma para conectar dispositivos IoT na nuvem, Permite executar funções de Lambda, manter sincronizados os dados dos dispositivos e comunicá-los de maneira segura, mesmo sem conexão direta com a nuvem) 9 - Desenvolvimento e ferramenta DevOps (Serviço de integração e entrega contínua (CI/CD), Serviço de build automatizado para projetos de software) 10 - Migração e transferência (Ferramenta para migrar o banco de dados para a AWS, Dispositivo físico para transferir grandes bancos de dados para AWS)



Google Cloud




Dentre as três apresentadas o Google Cloud é conhecido por sua forte inovação, tendo foco na área de big data, IAs e machine learning. Por conta disto o Google Cloud é o favorito em áreas especificas como serviços de dados e ferramentas de desenvolvimento. Seus principais serviços incluem: Google App Engine: Plataforma PaaS que permite desenvolver e escalar apps. Google Compute Engine: Máquinas virtuais escaláveis que rodam no proprio data centar do Google. Cloud Functions: Permite a execução de códigos em resposta a algum evento, sem precisar gerenciar servidor. Google Virtual Private Cloud: Rede virtual que permite conexão entre recursos do Google Cloud. Cloud Load Balancing: serviço que distribui o tráfego de rede entre várias regiões do globo, melhorando o desempenho. Google Cloud Storage: Permite o armazenamento de grande quantidade de dados em nuvem. Banco de Dados: CloudSpanner, Firestore, BigQuery e Cloud Bigtable são alguns dos serviços de banco de dados compatível com PostgreSQL, SQL Server e MySQL. Estes em conjunto garantem grande variedade de como você poderá usar o banco de dados dentro do Google Cloud. IAs e Machine Learning: Google cloud possui serviços de alto nível como AI Plataform, que permite criar treinar e implantar modelos de machine learning. Vision AI serviço que permite integrar análise de texto, reconhecimento de imagem e conversão de fala em texto. Vertex AI permite criar implanter e gerenciar modelos de ML. AutoML ferramenta que ajuda programadores com pouca experiencia a criar e personalizar modelos de ML. Sobre o preço do serviço do Google Icloud ele também possui sistema de pagamento parecido com o Azure, onde você paga pelo serviço que está usando. O diferencial é que no Google Cloud você começa com 300$ em créditos para usar na plataforma.



Azure




Plataforma de computação em nuvem distribuída pela Microsoft, possui boa amplitude de serviços como máquinas virtuais, banco de dados, armazenamento, redes, IAs, desenvolvimento de apps entre outros. Como o Azure é da Microsoft, existem facilidades quanto ao uso de integração com serviços da mesma. Integração com Windowns Server, Office 365 e Azure Arc são exemplos disso. Outra ferramenta de destaque é o Azure Site Recovery, que permite recuperação de máquinas virtuais e servidores em caso de algum desastre. No quesito financeiro, você pode abrir gratuitamente uma conta de teste válida por 30 dias, após o periodo você entra em um sistema onde você paga pelo que usar dentro dos, já citados, serviços disponíveis.
